# Proyecto Sprint 7 - Dashboard de Anuncios de Coches

Aplicación web desarrollada con Streamlit que permite explorar el conjunto de datos de anuncios de venta de coches usados (`vehicles_us.csv`).

## Funcionalidad

- Casilla de verificación para mostrar un histograma de la distribución del kilometraje (odómetro) de los vehículos.
- Casilla de verificación para mostrar un gráfico de dispersión que muestra la relación entre el kilometraje y el precio de los vehículos.
- Las casillas son mutuamente exclusivas: al seleccionar una se desactiva la otra automáticamente.

## Tecnologías utilizadas

- Python
- Pandas
- Plotly
- Streamlit

## Cómo ejecutar localmente

pip install -r requirements.txt
streamlit run app.py
